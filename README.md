# Hand Gesture Recognition (VGG16)

Lightweight hand gesture recognition project built with transfer learning on VGG16.

## What it does
- Classifies hand gestures into **5 classes**
- Uses **fine-tuned VGG16** for image classification
- Includes training and inference workflows in Jupyter notebooks

## Tech stack

- TensorFlow / Keras
- OpenCV
